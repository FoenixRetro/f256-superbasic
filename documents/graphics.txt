Graphics Module
---------------

Doesn't clip H/V (V clipped on Char/Sprite Draw)
Keeps track of the last two points.

Default bitmap $10000 (must be on 8k boundary)
Default GLUT 0
Default height 240

Graphics Command passed in AXY.

cccccccx xxxxxxxx yyyyyyyy

cccccccc

	0 BITMAP (X address, Y page)
	1 SPRITE (X on/off flag)
*	2 CLEAR (X colour)
*	3 COLOUR (X fgr, mode bits 7..0)
			Mode Bits: vflip|hflip|s2|s1|s0|-|m1|m0
			scale is (s2s1s0+1)
	4 DRAWC xxxxxxx character (advances horizontally)
	5 DRAWS sprite ID xxxxxx

*	16 MOVE 			
*	17 LINE 	
*	18 RECT
*	19 Fill RECT
*	20 CIRCLE
*	21 Fill CIRCLE
	22 ELLIPSE
	23 Fill ELLIPSE
	24 PLOT point
