Graphics Module
---------------

Doesn't clip H/V.
Keeps track of the last two points.

Default bitmap $10000 (must be on 8k boundary)
Default GLUT 0
Default height 240

Graphics Command passed in AXY.

cccccccx xxxxxxxx yyyyyyyy

cccccccc

0 CLEAR (X colour)
1 COLOUR (X fgr, Y mode)

16 MOVE 		
17 LINE 	
18 RECT
19 Fill RECT
20 ELLIPSE
21 Fill ELLIPSE 
22 DRAWC xxxxxxx character yyyyyyyy scale (from font memory),moves to next horizontal position (hflip/vflip option in upper bits of Y)
23 DRAWS sprite xxxxxxx yyyyyyy x:y forms a x 64 sprite address (hflip/vflip option)
